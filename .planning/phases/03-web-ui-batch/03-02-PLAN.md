1. 创建 `apps/web/src/hooks/useFormValidation.ts`：
   - 泛型 hook：`useFormValidation<T>(schema: z.ZodSchema<T>, initialValues: Partial<T>)`
   - 状态管理：values（表单值）、errors（字段级错误 Record<string, string>）、touched（已触碰字段 Set）
   - `setField(name, value)` — 更新值，如果字段已 touched 则立即验证该字段
   - `touchField(name)` — 标记字段为 touched，触发该字段验证
   - `validate()` — 全量验证，返回 `{ valid: boolean; data?: T; errors?: Record<string, string> }`
   - `reset(values?)` — 重置表单
   - 字段验证逻辑：用 schema.safeParse 验证整个对象，提取对应字段的错误信息
   - 注意：zod 的 error.issues 中 path 数组对应字段名，提取 message 作为错误文本
   - 安装 zod 到 apps/web 依赖：`pnpm --filter @sydev/web add zod`

2. 创建 `apps/web/src/components/config/FormField.tsx`：
   - 通用表单字段包装组件
   - Props：label（中文标签）、name、error（错误信息）、required、description（字段说明）、children（表单控件）
   - 渲染：Label + children + 错误信息（红色文字，带 AlertCircle 图标）+ 描述文字（灰色小字）
   - 使用 shadcn/ui 的 Label 组件
