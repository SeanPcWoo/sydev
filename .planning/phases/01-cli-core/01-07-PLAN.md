   - type: 'input'
   - message: 'Git 分支:'

   - type: 'input'
   - message: '项目名称:'
   - default: 从 git 仓库 URL 提取（例如 https://github.com/user/my-project.git → my-project）
   - validate: 3-50 字符，仅字母数字下划线连字符

如果选择"新建工程"：
1. 输入项目名称（name）- 无默认值

3. 选择构建类型（type）- 保持现有 8 种选项

**执行创建**：调用 RlWrapper.createProject()，不传递 version 参数
